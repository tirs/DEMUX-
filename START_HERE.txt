================================================================================
TYPICAL WORKFLOW
================================================================================

1. Upload audio file (WAV, MP3, FLAC, OGG) - Max 500MB
2. System processes through 4 stages:
   - Audio Separation (main bottleneck)
   - Harmonic/Percussive Decomposition
   - Composite Track Creation
   - Normalization
3. Download individual tracks or all as ZIP
4. Use tracks in your project

Processing time:
- 3-minute song: ~5 minutes (CPU)
- 3-minute song: ~2 minutes (GPU, if available)

================================================================================
ADDING YOUR OWN PROCESSORS
================================================================================

Create new audio processor:

1. Create class in core/processors.py extending PipelineStage
2. Implement execute() and validate_input() methods
3. Add to pipeline in api/app.py: pipeline.add_stage(NewStage())

Example in EXAMPLES.md

Swap audio models:

1. Create class extending SeparatorModel
2. Register: SeparatorFactory.register_separator("mymodel", MyModel)
3. Use: pipeline.add_stage(SeparationStage(separator_type="mymodel"))

No existing code needs to change!
